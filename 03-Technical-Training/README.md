# 🛡️ Technical Training

## Overview

The **CySecK Cybersecurity Finishing School** provided intensive, hands-on technical training designed to bridge the gap between academic learning and real-world cybersecurity practices. Through instructor-led sessions, practical laboratory exercises, live demonstrations, and guided assessments, I developed a strong foundation in offensive and defensive cybersecurity while gaining practical experience with industry-standard security tools and methodologies.

---

# 📚 Learning Methodology

The training emphasized experiential learning through:

- 💻 Hands-on laboratory exercises
- 🛡️ Live security demonstrations
- 🔍 Practical vulnerability assessments
- 🏆 Capture The Flag (CTF) preparation
- 📖 Case studies and technical discussions
- 📝 Security documentation and reporting
