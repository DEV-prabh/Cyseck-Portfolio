# 💻 Machine Write-ups

## Overview

As part of the CySecK Cybersecurity Finishing School, I completed multiple hands-on penetration testing laboratories designed to simulate real-world attack scenarios. These machine-based exercises focused on developing practical skills in reconnaissance, enumeration, exploitation, privilege escalation, and post-exploitation within controlled environments.

Each machine required a structured methodology, beginning with information gathering and progressing through vulnerability identification, exploitation, and documentation of findings.

---

# 🎯 Objectives

The machine assessments were designed to:

- Apply offensive security methodologies in practical environments.
- Develop systematic enumeration techniques.
- Identify and validate security vulnerabilities.
- Perform controlled exploitation.
- Practice privilege escalation techniques.
- Improve technical documentation and reporting.

---

# 🔍 Methodology

Every machine followed a structured penetration testing workflow:

1. Reconnaissance
2. Enumeration
3. Vulnerability Analysis
4. Exploitation
5. Privilege Escalation
6. Flag Capture
7. Documentation
