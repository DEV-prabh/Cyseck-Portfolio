# 🕵️ Digital Forensics

## Overview

As part of the CySecK Cybersecurity Finishing School, I completed a series of hands-on digital forensic investigations covering evidence acquisition, system analysis, metadata examination, multimedia forensics, mobile forensics, network analysis, and incident investigation.

The practical assessments focused on applying industry-standard forensic tools and structured investigation methodologies to analyze digital evidence while maintaining forensic integrity.
